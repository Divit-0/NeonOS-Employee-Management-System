# 🚀 NeonOS Employee Management System

A modern full-stack Employee Management System built using the MERN stack with JWT authentication, employee CRUD operations, analytics dashboard, Excel export, responsive mobile UI, and futuristic glassmorphism design.

---

# ✨ Features

✅ JWT Authentication  
✅ Login & Register System  
✅ Protected Routes  
✅ Employee CRUD Operations  
✅ Search Employees  
✅ Analytics Dashboard  
✅ Responsive Sidebar Navigation  
✅ Mobile Responsive Design  
✅ Framer Motion Animations  
✅ Excel Export Functionality  
✅ MongoDB Atlas Integration  
✅ REST API Architecture  

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Framer Motion
- Axios
- Recharts
- XLSX
- React Icons

## Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs
